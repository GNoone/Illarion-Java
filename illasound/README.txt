
   Illarion Sound Engine

Objectives

   The Illarion sound engine is the interface the different methods of audio
   playback that are used by the Illarion applications. Its objective is to
   provide a unified interface to the different playback methods for the
   applications.
   
Details

   The currently working implementation is the playback using JavaX. There is
   another implementation using OpenAL with the LWJGL library in work.


   Illarion Graphics Engine

Objectives

   The Illarion graphics engine is the interface to the OpenGL Bindings that
   are used by the Illarion applications. Its objective is to provide a
   unified interface to the different OpenGL bindings are ensure efficient
   rendering operations for the different applications.
   
Details

   The OpenGL bindings that are currently in use are JOGL and LWJGL. The
   libraries are distributed along with the Illarion Common Library.

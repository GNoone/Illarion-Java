
   Illarion Input Engine

Objectives

   The Illarion sound engine is the interface to the different input methods
   that are used by the Illarion applications. Its objective is to provide
   a unified interface to access the different sound systems.
   
Details

   The current implementations are using the LWJGL library for the input
   handling, the AWT framework or the NEWT framework.
